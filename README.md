# Cobalt: Real-Time Paper Trading Platform
A modern, real-time paper trading and portfolio management application designed to let users test and validate complex trading strategies without financial risk. Provides built-in candlestick visualization and simulate trades with real-world stock data.

# Architecture

**Frontend**	React.js, Next.js, JavaScript	

**Backend**	Node.js, Python (Flask)

**Storage & OAuth**	GCP (Firebase, Google Identity Platform)
